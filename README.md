# 🎮 CONTRAL GAME HUB
> **Play. Build. Dominate.**

This is the official repository for the **Contral Gaming Portal**, a high-performance web platform hosting 55+ titles. Featuring a futuristic neon-green aesthetic, Firebase integration, and a dynamic "Virtual Folder" system.

---

## 🚀 Quick Features
* **55+ Games:** From *Minecraft 1.12* to the full *Vex* and *Riddle School* series.
* **Virtual Folders:** Smart categorization (Action, IO, Minecraft) built via JavaScript.
* **Owner Bypass:** Secret "Access Code" system (e.g., `parker`) for instant entry.
* **PC Filter:** Toggleable visibility for PC-only titles vs. iPad-friendly games.
* **GitHub Powered:** All game assets (`.jpg`, `.png`) are hosted on GitHub for 0% lag.
